This is a copy of Mark Pilgrim's encoding detection package `chardet`.

Chardet used to be hosted at Google Code [1] but is no more, so this 
repo contains an unpacked chardet-1.0.1.tar.gz copied from PyPI [2]
on October 15, 2011.

[1] http://code.google.com/p/chardet/ (403 forbidden as of 2011-10-15)
[2] http://pypi.python.org/pypi/chardet
